Porjects based on c# scripting

FORAY

Foray is a 3D game for the android platform with first person character controls. The game is developed using Unity 3D and C# scripting. In this game, player has to collect a certain number of coins to complete different levels. There are enemies chasing the player and also moving platforms. Download the project and tweak some values to get your kind of gameplay.

Super Sparty Bros

This is a 2D game open to any platform and was developed on Unity 3D. The player collects the coins, escapes enemies and gets the cross, this is the whole idea. There are several levels in the game. Go check it out, if you are a 90's kid you might feel nostalgic.

  • Check it out

Zombie-Magic Walk

Zombie-Magic Walk is an android based application using the Augmented Reality Technology and also the concept of a virtual button. An animation of our zombie character will be triggered when we hover our hands over a virtual button in front of the camera. The application was developed using Unity 3D and vuforia API. Go check it out. It is fun!

  • Check it out

Box Buster

This is a first person character PC game developed using Unity 3D with full mouse controls to look around and keyboard controls to move around. In this game, player has to shoot certain boxes that will pop up randomly around the player. The player is always on a clock and shooting the correct boxes will buy him time but shooting the wrong boxes will result in a loss of time.

  • Check it out

The Gift is an augmented reality application which was exclusively built for Android platforms using Unity 3D game engine and vuforia API. The application was created as a gift to my little sister on her 17th birthday. Click on the button below to take a look at the project.

be it liberty

This is an android based Augmented Reality application which was developed using Unity 3D and vuforia API. The application access the camera of any android device and looks for a certain predefined image. Whenever that image comes up on screen, the application will portray a 3D model of "Statue of Liberty" on the target image. Click the button below to get to the project.

  • Check it out

Know what you got

Know what you got is an Augmented Reality application developed specifically for the android platform which displays the information about any object when scanned through the phone camera. The application uses google reverse image search API and is developed on Unity 3D and vuforia.

  • Check it out

Hot Deals

Hot Deals is a static application for the shopping of daily products. It is based on C# scripting and is made using Unity engine. The app is static but it includes a login system which retains the user credentials even after the app is restarted.

Tom and Jerry

Tom and Jerry is a fun chasing 2D game developed in core JAVA. The game was developed using Eclipse which is widely used as an IDE for JAVA. Here, different enemies strive to take down the player but our player, i.e.-Jerry needs to avoid each and every one of them. The more our player survives, the more he scores. So the game is all about trying to beat his or her own high score. It's fun !! Go check it out.
